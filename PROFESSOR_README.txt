📦 SETUP INSTRUCTIONS FOR PROFESSOR:

This is a standard Vite React application. To run:

1. Extract the ZIP file to a folder
2. Open terminal in that folder
3. Run: npm install
4. Run: npm run dev
5. Open browser to: http://localhost:5173

📋 REQUIREMENTS:
- Node.js (version 14 or higher)
- Modern web browser

🚀 STANDARD VITE REACT STRUCTURE:
✅ Standard package.json with Vite React dependencies
✅ Standard src/ folder structure:
   - main.jsx (React entry point)
   - App.jsx (Main game component)
   - App.css (Game styling)
   - index.css (Base styles)
✅ Standard index.html for Vite
✅ Standard vite.config.js
